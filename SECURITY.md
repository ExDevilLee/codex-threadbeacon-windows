# 安全说明

## 报告安全问题

请不要在公开 Issue 中提交包含任务标题、会话内容、本机路径、数据库内容、Token、Cookie、
凭据或其他个人数据的报告。

优先使用仓库 Security 页面的 **Report a vulnerability** 私密报告入口。若该入口不可用，
请只提交不含敏感数据的最小复现，并注明需要私密沟通。

## 支持范围

ThreadBeacon 当前是 Windows 11 技术预览版，只支持 README 中列出的本机 Codex 数据源。
只有本仓库 GitHub Releases 中的资产属于项目公开预览包。当前 EXE 尚未使用商业代码签名
证书，Windows SmartScreen 可能显示来源提示；请核对仓库、标签和 Release 页面后再运行。

ThreadBeacon 不要求管理员权限，不应请求关闭 Windows 安全功能，也不应要求上传 Codex
数据库、rollout、完整日志或会话内容。
